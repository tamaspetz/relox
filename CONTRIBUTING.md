# Contributing to relox

All contributions are welcome and highly appresciated.

## Local testing

We encourage you to test your changes locally before submitting them.

```sh
# Run all tests.
make test

# Run all tests and collect code coverage.
make test-coverage

# Format code.
make fmt
```
